# Cities

Named waypoints that players can teleport to for a points cost (or free).

## What you see

| Column | Notes |
|--------|-------|
| City Name | |
| Cost | Points, or "Free" |
| Position | Code format (`x, y, z`) |
| View Direction | |

## Actions

- **Create / Edit** a city. Fields: name, cost, position, view direction.
- **Teleport player** — pick an online player, moves them to the city.
- **Delete** a city.

## Notes

- Free cities (cost `0`) are good for spawn, trader hubs, and safe rooms.
- Paid cities are how you put a points sink in place for long-distance fast travel.
- The view direction sets which way the player faces on arrival, which matters for avoiding "you just teleported into a wall facing away from the door" moments.
