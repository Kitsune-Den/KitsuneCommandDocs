# Homes

A read-only view of every player's set home locations, with admin-side teleport and delete.

## What you see

| Column | Notes |
|--------|-------|
| Player Name | |
| Home Name | A player can have multiple named homes |
| Position | |

Search filters across both player name and home name.

## Actions

- **Teleport** — sends the home's owner to that home.
- **Delete** — removes the home.

## Notes

- Players set homes **in-game** via chat commands; there's no admin-side creation UI here. This page is for cleanup and occasional rescue teleports.
- Deleting a home doesn't refund any points the player spent to set it (if your server charges for that).
