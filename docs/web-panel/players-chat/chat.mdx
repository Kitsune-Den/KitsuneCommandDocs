# Chat

Server chat history with search, filtering, and lazy-loaded scrollback.

## What you see

| Column | Notes |
|--------|-------|
| Chat Type | Global, friends, party, whisper |
| Sender | Player or system |
| Timestamp | |
| Message | The text itself |

A search bar filters across the message text. A chat type dropdown narrows the feed to a specific channel.

## Actions

- **Search messages** by text.
- **Filter by chat type.**
- **Scroll up** to load older messages (lazy-loaded in pages).
- **Toggle auto-scroll** to pause or resume following new messages.
- **Send a message** as the panel (admins and higher). Viewers can read but not send.

## Real-time

New messages append to the feed as they arrive. Older messages load when you scroll up past the currently loaded range.

## Notes

- Auto-scroll stays off if you've manually scrolled up, so you can read history without the feed jumping back to the bottom.
- Sent messages appear in-game as server messages, not as the originating admin's player character.
