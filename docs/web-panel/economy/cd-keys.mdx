# CD Keys

One-time and multi-use redemption codes for items, points, or commands.

## What you see

| Column | Notes |
|--------|-------|
| Key Code | |
| Redemption Count | `x of max` |
| Expiry Date | Danger tag if expired |
| Description | |

## Actions

- **Create a key.** Pick what it grants (items, points, commands), set a max redemption count, and an expiry date.
- **Edit** an existing key's properties.
- **Redeem.** Requires picking an **online** player. Redemption is recorded in the Redemptions log.

## Redemption history

A second page under the CD Keys area shows the full redemption log:

| Column | Notes |
|--------|-------|
| Redemption Date | |
| Key ID | |
| Player Name | |
| Player ID | |

Filter by key ID to see everyone who's redeemed a specific code.

## Notes

- Expired keys are still listed but tagged in red; redeem attempts on them are rejected.
- If a player isn't online, the redeem dialog won't let you pick them. This keeps deliveries from silently failing.
