# VIP Gifts

Recurring or one-time reward packages assigned to specific players.

## What you see

| Column | Notes |
|--------|-------|
| Player Name | |
| Player ID | |
| Gift Name | |
| Claim Period | One-time, daily, weekly, monthly |
| Status | Ready (green) / Cooldown (orange) / Claimed (gray) |
| Total Claims | |
| Last Claimed | |

## Actions

- **Create a gift assignment.** Pick a player (either paste an ID or use the online-player picker), choose the reward package, and set the claim period.
- **Edit** an assignment's display name.

## Notes

- Status is derived from the claim period: a daily gift that was claimed today shows **Cooldown** until the next day rolls over.
- A warning appears if the player isn't online when they try to claim. The gift stays claimable until it's actually claimed.
- One-time gifts transition from Ready → Claimed permanently once used.
