# Web Panel

The web panel is a Vue 3 single-page app served from the mod's `wwwroot/` directory. It talks to the backend over HTTP (JSON) and WebSockets (real-time events) and defaults to **port 8890**.

This section walks through each major area from an admin's point of view. For the underlying HTTP surface, see [REST API Reference](../api-reference/index.mdx).

## Real-time and polling

A few screens are wired to WebSockets (`Console`, the Dashboard's Blood Moon progress). Most others poll the backend on a timer when they're in view. Both are called out on each page.

## Permissions

Actions that modify server state (kick, ban, give items, run commands, restore backups) are gated by role. A read-only account can browse everything but buttons that would write are hidden or disabled.
