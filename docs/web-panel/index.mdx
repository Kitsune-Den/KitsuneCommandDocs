# Web Panel

The web panel is a Vue 3 single-page app served from the mod's `wwwroot/` directory. It talks to the backend over HTTP (JSON) and WebSockets (real-time events).

This section walks through each major area of the panel from a server-admin's point of view. For the underlying HTTP surface, see [REST API Reference](../api-reference/index.mdx).

## Sections

- **Dashboard & Monitoring** — Dashboard, Map, Console
- **Players & Chat** — Players, Chat, Tickets, Colored Chat
- **Economy** — Points, Store, CD Keys, VIP Gifts, Purchase History
- **Teleportation** — Homes, Cities, History
- **Game Events** — Blood Moon Vote, Task Scheduler
- **Content** — Item Database
- **Server Management** — Server Control, Config Editor, Mods, Backups, Updates

Individual pages for each area are being written. For now, the panel's UI labels line up with the backend controllers documented under [REST API Reference](../api-reference/index.mdx).
