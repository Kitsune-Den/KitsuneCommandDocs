# Introduction

**KitsuneCommand** is an open-source mod for 7 Days to Die dedicated servers that provides a RESTful API and a modern web management panel. It runs on both Windows and Linux, and is a clean-room V2 rewrite of [ServerKit](https://github.com/IceCoffee1024/7DaysToDie-ServerKit) with a Vue 3 frontend and improved security.

Three pillars:

- **Monitoring.** Live dashboards, GPS map with player tracking, web console with log streaming.
- **Management.** Player tools, chat bridge, economy, teleports, scheduler, mods, backups, config editor.
- **Map.** Leaflet-based world map rendered in SkiaSharp, updated in real time over WebSockets.

## What's in the box

| Area | Highlights |
|------|-----------|
| Dashboard & Monitoring | Real-time stats (FPS, memory, day/time), Leaflet map, web console with history |
| Players & Chat | Player list with inventory/skills, kick/ban, chat history, colored names |
| Economy | Points, configurable store, CD keys, VIP gifts (daily/weekly/monthly), purchase history |
| Game Systems | Teleports (home/city/friend), Blood Moon voting, task scheduler, item database |
| Server Admin | Server control, `serverconfig.xml` editor, mod manager, auto-backups, plugin system |
| Localization | English, Japanese, Korean, Chinese Simplified, Chinese Traditional |

## Tech stack

- **Backend:** C# 11 on .NET Framework 4.8, OWIN + ASP.NET Web API 2, Autofac DI, SQLite via Dapper
- **Frontend:** Vue 3, TypeScript 5, Vite 6, PrimeVue 4
- **Real-time:** WebSocketSharp
- **Auth:** OAuth2 with BCrypt
- **Map tiles:** SkiaSharp + Leaflet
- **Game integration:** Harmony runtime patching
